# 🚀 Design Patterns in C# (.NET)

A collection of commonly used **Design Patterns** implemented in C# using .NET.
This repository demonstrates how to write clean, maintainable, and scalable code by applying well-known software design patterns.

---

## 📌 Overview

This project includes practical implementations of multiple **Design Patterns**, each in its own module with a simple and clear example.

The goal of this repository is to:

* Improve understanding of design patterns
* Demonstrate real-world usage in C#
* Help developers write better structured code

---

## 🧠 Implemented Design Patterns

### 🏭 Factory Method Pattern

* Creates objects without specifying the exact class
* Promotes loose coupling

---

### 🧍 Singleton Pattern

* Ensures a class has only one instance
* Provides a global point of access

---

### 🎯 Strategy Pattern

* Allows selecting an algorithm at runtime
* Encapsulates interchangeable behaviors

---

### 🔌 Adapter Pattern

* Converts one interface into another
* Helps incompatible classes work together

---

### 🎁 Decorator Pattern

* Adds behavior to objects dynamically
* Follows Open/Closed Principle

---

## ⚙️ Setup & Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yosif-taha/Design_Patterns.git
cd Design_Patterns
```

---

### 2️⃣ Open the Solution

Open the `.sln` file using:

* Visual Studio
  or
* VS Code

---

### 3️⃣ Run the Project

Each pattern is implemented as a separate example.

Run any project individually:

```bash
dotnet run
```
